# Packaging and Distribution

Packaging is a core PyNative UI feature.

## Planned Targets

| Platform | Output |
| --- | --- |
| Windows | `.exe` or installer |
| Android | `.apk` and later `.aab` |
| Linux | AppImage or `.deb` |
| macOS | `.app` or `.dmg` |
| iOS | `.ipa` |

## Planned Commands

```powershell
pynative build windows
pynative build apk
```

These commands are planned for future phases. Phase 0 focuses on project structure and the Rust/Python bridge.
